# Tiny Renderer

This tiny ray-tracing renderer was created as part of my bachelor’s thesis, which aimed to explain the fundamentals of ray tracing, light transport, and the underlying physics. It also covered key ray-tracing algorithms and introduced basic BRDFs. The thesis was awarded an A.
